Introduction to Web Development

General Idea Web Development

Web development is the method of creating websites where you will have to create many webpages and link all at once. Web development is basically divided into two parts:

  • Frontend: This is the part of the website that can only be seen by the user.

  • Backend: This is the part of the website that saves data of the website and this part can only be seen by the owner of the website.

HTML Basics

HTML is an abbreviation for [Hyper Text Markup Language]. This is commonly used to provide contant on your webpage. HTML is the standard language for building a website. In a short term, this provides the skeleton of the website. A html file can be saved with the extension in the file name [.html] & the file type as [All files]. A css file must be linked to the html document, and in the head of the html doc using the element['link'], which will allow the css file to have effects on your html documents. In this link, there is an attribute [rel] which stands for relationship, this will be set to "stylesheet" and there is also another attribute [href] This attribute specifies the URL of the linked resource. A URL can be absolute or relative. This can also be set to the name of your css file

Here is the basic structure of a html document.

Discussion

  • '!DOCTYPEhtm' : Shows the type of document that's written and also shows the version of html.
  • "head" : this tells the browser all of the necessary information of the html file.

CSS Basics

CSS (Cascading Style Sheets) is used to style and layout web pages. It allows you to apply styles to HTML elements and control the layout of multiple web pages all at once. A css file can be saved with the extension in the file name [.css] & the file type as [All files].

CSS syntax

A CSS rule-set consists of a selector, curly braces, and a declaration block:

There are four types of selectors, and they are:

  • Element Selector: This selects all html elements of the same kind.
  • Class Selector: this selects two or more elements of the given class.

    • In the html file, the class is given like this.

    In the css file we can design our element using its class, Id, attributes.

  • Id Selector: this selects a specific element of the given Id

    • In the html file, this is how we give an element an Id.

    In the css file, we design an element using its Id like this.

  • Attribute Selector: This selects a specific element with the given attribute.

    • In the html file, this is how we attach an attribute to a specific element.

    In the css file, this is how we apply changes to our attributed element.

    HTML and CSS Code Example

    HTML File (index.html)

    
<!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <meta name="viewport" content=" 
     width=device-width, initial-scale=1.0">
    <title>Simple Web Page</title>
    <link rel="stylesheet" 
     href="styles.css">
</head>
<body>
    <h1>Welcome to My Web Page</h1>
    <p>This is a simple example of an 
     HTML and CSS file for beginners.</p>
    <button class="styled-button"> 
     Click Me!</button>
</body>
</html>

    CSS File (styles.css)

    
/* This is a comment in CSS */

body {
    font-family: Arial, sans-serif;
    margin: 0;
    padding: 20px;
    background-color: #f0f0f0;
}

h1 {
    color: #333;
}

p {
    font-size: 16px;
    line-height: 1.5;
    color: #666;
}

.styled-button {
    background-color: #4CAF50; /* 
     Green background */
    color: white; /* White text */
    padding: 10px 20px; /* Some padding */
    border: none; /* Remove borders */
    border-radius: 5px; /* Rounded corners */
    cursor: pointer; /* Pointer/hand icon */
    font-size: 16px; /* Increase font size */
}

.styled-button:hover {
    background-color: #45a049; /* 
     Darker green */
}
